1. What Oslo Does

Oslo is a chess puzzle training bot. It allows users in a Discord server to:

  • Request and solve chess puzzles from the Lichess open database
  • Earn points based on puzzle difficulty and solve quality
  • View personal statistics and compete on server and global leaderboards
  • Filter puzzles by difficulty level and tactical theme

2. Admin Commands

Oslo includes a small set of admin-only commands restricted to the bot developer. These commands are entirely about Oslo's own operation and have no ability to affect your server, its members, its channels, or its settings in any way.

Admin commands include:

  • !sleep / !wake — temporarily pause or resume Oslo's puzzle responses globally
  • !adminstatus — view Oslo's operational stats (active puzzles, total solves, database status)
  • !leaveserver — removes Oslo from server in case of violation of acceptable use
  • !exportdata — export Oslo's own database for backup or migration purposes

None of these commands can delete messages, modify roles, ban or kick members, alter channel settings, or interact with your server in any structural way. Oslo's permissions are intentionally minimal — it only needs to view channels, send messages, attach files, and read message history in the channels it's used in.
